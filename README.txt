Steps to run:

1)run pdollar.exe
2)In the same directory, open cmd
3)type:"pdollar" for menu
4)type:"pdollar -t arrowhead.txt" for adding gesture set
5)type:"pdollar arrowhead_eventfile.txt" to recognize
6)type:"pdollar -r" to remove gesture set

NOTE: Ensure the the gesture and event .txt files are 
in the same folder as pdollar.exe

Reference PDollarGestureRecognizer.dll is added and taken from the C# source code provided on the link:
http://depts.washington.edu/madlab/proj/dollar/pdollar.html
